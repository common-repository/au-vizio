== Description ==

AU VIZIO enables improved management for designers and webmasters to manage affiliate banners by Serbian online t-shirt store VIZIOshop.com.

WARNING! Development abandoned!

= Features =

* Site affiliate ID included
* Designer username to link global banner to designers inline store included
* Customizable default price for all t-shirts
* Customizable multiple t-shirt groups
* Get proper random t-shirt from group by checking post keywords or/and post ID's
* Convertor to get properly formated data for t-shirt lists

== Installation ==

To install this plugin you can use old-school procedure:

1. Download `au-vizio.zip` archive and unzip it to temporary directory
1. Upload newly created `au-vizio` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the `Plugins` menu in WordPress
1. Setup plugin parameters in `VIZIOshop` part of `Settings`
1. Place widget to sidebar and injoy

or you can use automated installation through `Plugin Search`.
